## Descricao

Tela de central de notificacoes in-app: lista de notificacoes recebidas (alertas de
parametros, novidades, mensagens), marcacao de lida, badge de nao lidas no icone do sino.

## Criterios de Aceite

- [x] Badge de nao lidas no tab Home (icone Home no BottomTabs)
- [x] NotificationCenterScreen: lista de notificacoes com tipo, mensagem, tempo
- [x] GET /api/v1/notifications para lista
- [x] Tap na notificacao navega para tela relevante (AquariumDetail se alerta, etc.)
- [x] PATCH /api/v1/notifications/{id}/read ao tocar
- [x] Marcar todas como lidas: PATCH /api/v1/notifications/read-all
- [x] Deletar notificacao individual com swipe (Swipeable)
- [x] Badge desaparece quando todas lidas
- [x] Polling unread count a cada 60s

## Notas Tecnicas

- Badge count: GET /api/v1/notifications/unread-count (polling a cada 60s)
- Tipo de notificacao: `alert`, `message`, `tip`, `order_update`
