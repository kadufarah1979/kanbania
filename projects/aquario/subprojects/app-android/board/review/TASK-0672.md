## Descricao

Criar a tela de historico de parametros com grafico de serie temporal usando VictoryNative XL.
Seletor de periodo: 1h, 6h, 24h, 7d, 30d. Um grafico por parametro (temperatura, pH, TDS).

## Criterios de Aceite

- [x] Tela acessivel via botao "Historico" em AquariumDetailScreen
- [x] Seletor de periodo (chips): 1h | 6h | 24h | 7d | 30d
- [x] Grafico AreaChart para cada parametro (temperatura, pH, TDS)
- [x] Eixo X: timestamps formatados (hh:mm para intraday, dd/MM para multi-dia)
- [x] Skeleton enquanto carrega; EmptyState se sem dados no periodo
- [x] Scroll vertical entre os 3 graficos

## Notas Tecnicas

- Usar `VictoryNative XL` (`victory-native`)
- Dados: array de `{ timestamp: string, value: number }` por parametro
- Performance: nao renderizar mais de 500 pontos — agregar no backend via periodo
