Auditar todas as strings hardcoded no app e centralizar em arquivos de traducao pt-BR.
Preparar estrutura para suporte a outros idiomas no futuro (mesmo que so pt-BR seja entregue agora).

## Criterios de Aceite

- [x] Biblioteca: objeto simples de constantes (mais leve para o projeto atual)
- [x] Arquivo `src/shared/i18n/strings.ts` com todas as strings
- [ ] Zero strings hardcoded em componentes (exceto emojis e icones)
- [x] Nomes de parametros internalizados: "Temperatura", "pH", "TDS", "Amonia" etc
- [x] Formatos de data/hora: `dd/MM/yyyy HH:mm` (padrao brasileiro)
- [x] Formatacao de moeda: `R$ 1.234,56`
