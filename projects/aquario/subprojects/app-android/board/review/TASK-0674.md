## Descricao

Tela de gestao de regras de alerta do aquario. O usuario define limites personalizados
(ex: temperatura > 30°C = alerta critico). Listar regras existentes, criar nova regra,
editar e desativar/excluir.

## Criterios de Aceite

- [x] Tela acessivel via navegacao a partir de AquariumSettings
- [x] Lista de regras com parametro, operador, threshold e severidade
- [x] Formulario de criacao: parametro, operador (>, <, >=, <=), valor, severidade (warn/crit)
- [x] POST /api/v1/aquariums/{id}/alert-rules para criar
- [x] PATCH /api/v1/aquariums/{id}/alert-rules/{ruleId} para editar (toggle enabled)
- [x] DELETE /api/v1/aquariums/{id}/alert-rules/{ruleId} para excluir
- [x] Toggle de ativo/inativo por regra

## Notas Tecnicas

- Usar FlashList para a lista de regras
- Formulario com React Hook Form + Zod
