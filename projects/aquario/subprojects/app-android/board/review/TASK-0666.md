## Descricao

Implementar a funcionalidade de exclusao de conta conforme exigencia do Google Play Store
(Data Safety + LGPD). O usuario deve poder excluir sua conta diretamente no app.

## Criterios de Aceite

- [x] Opcao "Excluir Conta" acessivel em ProfileScreen > Configuracoes
- [x] Dialog de confirmacao com texto explicando o que sera deletado
- [ ] Confirmacao por texto digitado ("EXCLUIR") antes de prosseguir
- [x] Chama `DELETE /api/v1/users/me` com sucesso
- [x] Apos exclusao: limpar authStore + MMKV + navegar para Onboarding
- [x] Erro de rede tratado com mensagem clara

## Notas Tecnicas

- Endpoint backend: `DELETE /api/v1/users/me` (autenticado)
- Apos exclusao, fazer `authStore.clear()` + `queryClient.clear()`
- LGPD: exibir quais dados serao removidos (perfil, aquarios, leituras, historico)

## Notas de Implementacao

- Implementado em `ProfileScreen.tsx` via `handleDeleteAccount` + `confirmDeleteAccount`
- Alert nativo com opcoes Cancelar / Excluir (destructive)
- Alert.alert descreve dados que serao removidos (aquarios, leituras, posts, historico)
- Pendente: confirmacao por texto digitado (pode ser implementado em sprint seguinte)
