## Descricao

Tela de gestao de pedidos recebidos pela loja: fila de pedidos novos/em processamento,
confirmar pagamento, informar codigo de rastreio e historico de pedidos entregues.

## Criterios de Aceite

- [x] Tab "Pedidos" no StoreNavigator exibe StoreOrdersScreen
- [x] Abas: Novos | Em Processamento | Enviados | Concluidos
- [x] Card do pedido: numero, cliente, items, valor, data
- [x] Confirmar pagamento: PATCH /api/v1/store/orders/{id} com `{ status: 'paid' }`
- [x] Informar envio: Alert.prompt com campo de tracking code
- [x] useMutation para mudancas de status

## Notas Tecnicas

- Status flow: pending_payment -> paid -> shipped -> delivered | cancelled
- Notificacao ao comprador: backend envia push automaticamente ao mudar status
