## Descricao

Tela de configuracoes do aquario com opcoes de editar nome, volume, tipo de agua,
limites de parametros (temperatura min/max, pH min/max, TDS min/max) e desvincular dispositivo.

## Criterios de Aceite

- [x] Acessivel via icone de engrenagem em AquariumDetailScreen
- [x] Campos editaveis: nome, volume, tipo de agua
- [x] Secao de limites de parametros (temperatura, pH, TDS) — min e max
- [x] PATCH /api/v1/aquariums/{id} ao salvar
- [x] Opcao "Desvincular Dispositivo" com confirmacao
- [x] DELETE /api/v1/aquariums/{id}/devices/{deviceId} para desvincular
