## Descricao

Tela de detalhe de produto do marketplace: galeria de fotos, descricao completa,
dados do seller, stock disponivel e botao "Adicionar ao Carrinho".

## Criterios de Aceite

- [x] GET /api/v1/marketplace/listings/{id} para dados do produto
- [x] Galeria de fotos horizontalmente scrollable
- [x] Nome, descricao, preco, estoque disponivel
- [x] Card do seller: nome, rating com estrela
- [x] Seletor de quantidade (stepper 1-N, max = stock)
- [x] Botao "Adicionar ao Carrinho" chama cartStore.addItem()
- [x] Badge do carrinho atualizado no botao

## Notas Tecnicas

- cartStore (Zustand): addItem, removeItem, updateQty, clear, clearCart
- CartItem: listingId, title, price, quantity, sellerId, imageUrl
- Nao faz POST para o backend ainda — so cartStore local
