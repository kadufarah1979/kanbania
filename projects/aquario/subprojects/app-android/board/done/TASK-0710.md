## Descricao

Gestao de colaboradores da loja: convidar funcionarios por email, definir nivel de
acesso (admin, operador, atendimento) e revogar acesso.

## Criterios de Aceite

- [x] Lista de colaboradores ativos com nome, email, nivel, data de entrada
- [x] Convidar: Alert.prompt com email + POST /api/v1/store/collaborators/invite
- [x] Alterar nivel: Alert.alert com opcoes + PATCH /api/v1/store/collaborators/{id}
- [x] Revogar acesso: DELETE /api/v1/store/collaborators/{id} com confirmacao
- [x] Niveis: admin (tudo), operator (pedidos + catalogo), support (so atendimento)

## Notas Tecnicas

- Convite por email: backend envia email com link para aceitar
- Colaborador aceita o convite logado no app com a conta dele
