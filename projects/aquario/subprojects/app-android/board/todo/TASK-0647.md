Inicializar o projeto React Native + Expo SDK 52 dentro do monorepo `/home/carlosfarah/Projects/aquario/mobile/`. Configurar toda a infraestrutura de toolchain antes de qualquer feature.

## Criterios de Aceitacao

- [ ] `mobile/` inicializado com `npx create-expo-app@latest` (TypeScript template)
- [ ] `app.config.ts` configurado: `scheme: 'aquabook'`, `name: 'AquaBook'`, `bundleIdentifier: 'app.aquabook.android'`, icone e splash placeholders
- [ ] `babel.config.js` com `babel-preset-expo` + plugin NativeWind
- [ ] `tsconfig.json` com paths: `@/*` -> `./src/*`
- [ ] `.env.example` com `API_BASE_URL`, `WS_BASE_URL`, `EXPO_PUBLIC_*` vars
- [ ] `eas.json` com profiles: `development` (internal dist), `preview` (APK interno), `production` (AAB Play Store)
- [ ] Estrutura de pastas criada conforme `docs/dev/APP_MOBILE.md` (src/app/, src/shared/)

- [ ] `npx expo start` executa sem erros no Metro
