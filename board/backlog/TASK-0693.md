## Descricao

Tela de historico de compras do usuario. Lista de pedidos com status (aguardando pagamento,
pago, enviado, entregue, cancelado). Detalhe do pedido com itens e tracking.

## Criterios de Aceite

- [ ] Acessivel em ProfileScreen > "Meus Pedidos"
- [ ] GET /api/v1/marketplace/orders?buyer=me para lista
- [ ] Card de pedido: numero, data, status com StatusBadge, valor total
- [ ] Tap abre OrderDetailScreen: itens, status, tracking code (se disponivel), botao de contato com seller
- [ ] Status: pending_payment, paid, shipped, delivered, cancelled
- [ ] Pull-to-refresh

## Notas Tecnicas

- Reutilizar StatusBadge para exibir status do pedido com cores adequadas
- Tracking code: link externo para Correios/transportadora se disponivel
