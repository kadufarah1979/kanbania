## Descricao

Auditar todos os security groups definidos no Terraform e verificar conformidade com PCI Req 1:
nenhuma regra `0.0.0.0/0` para portas sensiveis (DB, admin, SSH), segmentacao adequada entre tiers.

## Criterios de Aceite

- [ ] Levantamento completo de todos os security groups em `IaC/modules/security-groups/`
- [ ] Nenhuma regra allow `0.0.0.0/0` para portas 22 (SSH), 3306/5432 (DB), 27017 (MongoDB)
- [ ] Documentacao de cada regra com justificativa de negocio
- [ ] Segmentacao: tier de app nao acessa diretamente tier de DB via regras abertas
- [ ] `terraform plan` sem breaking changes apos restricoes aplicadas
- [ ] Relatorio de auditoria em `docs/sg-audit-2026-Q2.md`
