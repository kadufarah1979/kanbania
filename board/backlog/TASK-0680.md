## Descricao

Tela de perfil publico de outro usuario. Exibe avatar, nome, bio, badges, contagem de
aquarios/posts/seguidores e grid de posts publicos. Permite seguir/desseguir o usuario.

## Criterios de Aceite

- [ ] Acessivel via tap no avatar/nome em PostCard no feed
- [ ] GET /api/v1/users/{userId}/profile para dados do perfil
- [ ] Botao "Seguir" / "Seguindo" com toggle (POST/DELETE /api/v1/social/follows/{userId})
- [ ] Grid de posts publicos do usuario (FlashList 2 colunas)
- [ ] Badges exibidos em row scrollable
- [ ] Se for o proprio usuario: nao exibir botao seguir, link para editar perfil
