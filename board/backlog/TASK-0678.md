## Descricao

Implementar o Feed Social (tab Comunidade no UserNavigator) com FlashList de posts,
pull-to-refresh, infinite scroll, e acoes de like e comentario.

## Criterios de Aceite

- [ ] Tab "Comunidade" no UserNavigator exibe FeedScreen
- [ ] FlashList com posts (avatar, nome, foto opcional, texto, parametros do aquario opcional)
- [ ] Infinite scroll via cursor: GET /api/v1/social/feed?cursor=&limit=20
- [ ] Pull-to-refresh
- [ ] Botao de like: POST /api/v1/social/posts/{id}/like (toggle)
- [ ] Contagem de likes e comentarios visivel no card
- [ ] Tap no post abre PostDetailScreen com comentarios
- [ ] GET /api/v1/social/posts/{id}/comments para lista de comentarios

## Notas Tecnicas

- Usar FlashList (Shopify) — nao FlatList — para performance
- useinfiniteQuery do TanStack Query para paginacao por cursor
- Post card: componente reutilizavel PostCard em shared/components
