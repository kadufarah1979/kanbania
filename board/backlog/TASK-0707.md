## Descricao

Tela de gestao do catalogo de produtos da loja: lista de listings, criar novo produto
com fotos, editar preco/estoque e ativar/desativar listing.

## Criterios de Aceite

- [ ] Tab "Catalogo" no StoreNavigator exibe CatalogManagementScreen
- [ ] FlashList de produtos com foto, nome, preco, estoque, status (ativo/inativo)
- [ ] Criar produto: nome, descricao, categoria, preco, estoque, fotos (max 5)
- [ ] Upload de fotos: expo-image-picker multiplo, comprimir antes de enviar
- [ ] Editar: todos os campos
- [ ] Toggle ativo/inativo (PATCH /api/v1/store/listings/{id} com `{ active: bool }`)
- [ ] POST /api/v1/store/listings para criar

## Notas Tecnicas

- Upload: multipart/form-data com array de imagens
- Comprimir: max 1080px por imagem via expo-image-manipulator
