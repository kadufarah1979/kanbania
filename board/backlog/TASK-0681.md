## Descricao

Tela de gestao da fauna e flora do aquario. Lista todas as especies cadastradas com
foto, nome cientifico/popular, quantidade. Permite adicionar, editar e remover especies.

## Criterios de Aceite

- [ ] Tab "Habitantes" em AquariumDetailScreen (ou secao dedicada)
- [ ] FlashList de especies com foto miniatura, nome, quantidade
- [ ] Busca de especie: autocomplete da base de dados do backend
- [ ] Adicionar especie: nome cientifico/popular, quantidade, data de adicao, observacoes
- [ ] Editar: quantidade, observacoes
- [ ] Remover com confirmacao
- [ ] GET /api/v1/aquariums/{id}/livestock para lista
- [ ] POST, PATCH, DELETE /api/v1/aquariums/{id}/livestock/{speciesId}

## Notas Tecnicas

- Banco de especies: GET /api/v1/species?q=nemo (autocomplete)
- Foto da especie: servida pelo backend, com fallback de icone Fish
