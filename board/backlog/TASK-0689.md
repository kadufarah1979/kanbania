## Descricao

Permitir que o usuario compartilhe um link publico do seu aquario que pode ser visualizado
sem login (web ou deep link). Configurar a privacidade (publico/privado) nas configuracoes.

## Criterios de Aceite

- [ ] Toggle "Aquario Publico" em Configuracoes do Aquario
- [ ] PATCH /api/v1/aquariums/{id} com `{ is_public: bool }`
- [ ] Botao "Compartilhar" em AquariumDetailScreen (Share API do RN)
- [ ] Link: `https://aquabook.app/aquarium/{public_token}`
- [ ] Tela PublicAquariumScreen: parametros atuais + status + nome (sem login necessario)
- [ ] Deep link abre o app se instalado, senao abre no browser

## Notas Tecnicas

- `Share.share({ url, message })` do React Native
- Public token: retornado pelo backend ao tornar publico
- PublicAquariumScreen e acessivel sem autenticacao (nao usa apiClient autenticado)
