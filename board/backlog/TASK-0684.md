## Descricao

Implementar a tela de lista de conversas e a tela de detalhe de conversa (sem envio ainda).
Chat entre usuarios (DM) para suporte entre comunidade, e base para chat de consultoria.

## Criterios de Aceite

- [ ] Tab "Mensagens" no UserNavigator exibe ConversationListScreen
- [ ] GET /api/v1/chat/conversations para lista
- [ ] Lista com avatar, nome, ultima mensagem, horario, badge de nao lidos
- [ ] Tap abre ConversationDetailScreen com historico de mensagens
- [ ] GET /api/v1/chat/conversations/{id}/messages (paginado por cursor)
- [ ] FlashList de mensagens com baloes (proprio/outro), timestamps
- [ ] Indicador "lida" para mensagens enviadas pelo usuario

## Notas Tecnicas

- Mensagem: `{ id, sender_id, content, created_at, read_at? }`
- Reutilizar em PRO (chat consultoria, TASK-0702) e STORE (atendimento, TASK-0709)
