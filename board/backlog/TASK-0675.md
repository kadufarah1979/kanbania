## Descricao

Tela de registro de leituras manuais via test kit (aqueles que o usuario faz com reagentes).
Parametros: pH, Amonia (NH3), Nitrito (NO2), Nitrato (NO3), GH, KH, Salinidade, Fosfato.
Os valores sao registrados no historico junto com as leituras automaticas do sensor.

## Criterios de Aceite

- [ ] Tela "Test Kit" acessivel em AquariumDetailScreen
- [ ] Selecionar quais parametros medir (checkboxes)
- [ ] Campo numerico por parametro selecionado
- [ ] Data/hora: automatica (now) com opcao de ajustar
- [ ] POST /api/v1/aquariums/{id}/readings/manual com array de medicoes
- [ ] Historico de test kits: lista das ultimas 10 entradas manuais
- [ ] Feedback visual ao salvar

## Notas Tecnicas

- Endpoint: `POST /api/v1/aquariums/{id}/readings/manual`
- Body: `{ readings: [{ parameter: string, value: number, unit: string, measured_at: ISO8601 }] }`
