## Descricao

Implementar a UI de badges e conquistas: grid no perfil, animacao de desbloqueio
(modal com confete ao ganhar novo badge), e tela de todas as conquistas (ganhas/pendentes).

## Criterios de Aceite

- [ ] GET /api/v1/users/me/badges para badges do usuario
- [ ] GET /api/v1/badges para catalogo completo de badges
- [ ] Grid de badges no ProfileScreen: ganhos (coloridos) e nao ganhos (cinza)
- [ ] Tap em badge: modal com descricao + criterio para ganhar
- [ ] Animacao de desbloqueio: modal com confete (react-native-confetti-cannon) ao ganhar badge novo
- [ ] Tela "Conquistas" com progresso por categoria

## Notas Tecnicas

- `react-native-confetti-cannon` para animacao
- Verificar novos badges apos acoes relevantes (ex: apos criar 1o aquario, verificar badges)
