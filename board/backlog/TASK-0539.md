## Descricao

Criar o widget "Novidades do AquaBook" no dashboard dos usuários logados, exibindo
os 5 artigos mais recentes filtrados pelas categorias preferidas do usuário.

## Criterios de Aceite

- [ ] Widget `BlogNewsWidget` criado em `frontend/src/components/dashboard/`
- [ ] Busca 5 artigos via `GET /api/v1/blog/posts?limit=5` (com filtro de categorias do usuário)
- [ ] Cada item exibe: thumbnail pequeno (40x40), título (line-clamp-2), categoria (badge colorido), data relativa
- [ ] Link em cada item abre o artigo em `/blog/{slug}`
- [ ] Link "Ver todos os artigos" no rodapé do widget → `/blog`
- [ ] Loading skeleton enquanto carrega
- [ ] Estado vazio se nenhum artigo publicado ainda
- [ ] Revalidação automática a cada 30 minutos (SWR ou React Query)
- [ ] Inserido no layout do dashboard (coluna lateral ou seção inferior)
- [ ] Responsivo

## Contexto Tecnico

- Depende dos endpoints de blog (TASK-0534 fase de blog ou endpoint público já criado)
- Reutilizar sistema de badges de categoria de `docs/public-design-system.md`
