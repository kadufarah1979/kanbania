## Descricao

Criar a tela de historico de parametros com grafico de serie temporal usando VictoryNative XL.
Seletor de periodo: 1h, 6h, 24h, 7d, 30d. Um grafico por parametro (temperatura, pH, TDS).

## Criterios de Aceite

- [ ] Tela acessivel via botao "Historico" em AquariumDetailScreen
- [ ] Seletor de periodo (chips): 1h | 6h | 24h | 7d | 30d
- [ ] Grafico AreaChart para cada parametro (temperatura, pH, TDS)
- [ ] Eixo X: timestamps formatados (hh:mm para intraday, dd/MM para multi-dia)
- [ ] Eixo Y: valor com unidade, linhas de referencia em min/max de alerta
- [ ] Skeleton enquanto carrega; EmptyState se sem dados no periodo
- [ ] Scroll vertical entre os 3 graficos

## Notas Tecnicas

- Usar `VictoryNative XL` (`victory-native/src/cartesian/`)
- Dados: array de `{ timestamp: string, value: number }` por parametro
- Performance: nao renderizar mais de 500 pontos — agregar no backend via periodo
