## Descricao

Tela de artigos/blog curado com feed RSS traduzido para pt-BR (backend ja tem o blog_curator_service).
Lista de artigos com thumbnail, titulo, fonte e abertura via WebView interno.

## Criterios de Aceite

- [ ] Secao "Artigos" no UserNavigator (tab ou dentro de Comunidade)
- [ ] GET /api/v1/blog/articles com lista de artigos curados
- [ ] Card: thumbnail, titulo em pt-BR, fonte original, data
- [ ] Tap abre ArticleDetailScreen com WebView (expo-web-view) ou abertura externa
- [ ] Pull-to-refresh para atualizar feed
- [ ] Skeleton loading
- [ ] Favoritar artigo (salvar localmente em MMKV)

## Notas Tecnicas

- Backend: `GET /api/v1/blog/articles` (blog_curator_service — ja implementado)
- WebView: `expo-web-browser` (abertura externa) ou WebView nativo (in-app)
- Preferir abertura externa para evitar problemas de JS injection
