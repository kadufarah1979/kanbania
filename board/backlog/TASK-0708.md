## Descricao

Tela de gestao de pedidos recebidos pela loja: fila de pedidos novos/em processamento,
confirmar pagamento, informar codigo de rastreio e historico de pedidos entregues.

## Criterios de Aceite

- [ ] Tab "Pedidos" no StoreNavigator exibe OrdersManagementScreen
- [ ] Abas: Novos | Em Processamento | Enviados | Concluidos | Cancelados
- [ ] Card do pedido: numero, cliente, items, valor, data
- [ ] Confirmar pagamento: PATCH /api/v1/store/orders/{id} com `{ status: 'paid' }`
- [ ] Informar envio: modal com campo de tracking code + transportadora
- [ ] Cancelar pedido com motivo

## Notas Tecnicas

- Status flow: pending_payment → paid → shipped → delivered | cancelled
- Notificacao ao comprador: backend envia push automaticamente ao mudar status
