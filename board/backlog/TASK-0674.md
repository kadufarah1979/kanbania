## Descricao

Tela de gestao de regras de alerta do aquario. O usuario define limites personalizados
(ex: temperatura > 30°C = alerta critico). Listar regras existentes, criar nova regra,
editar e desativar/excluir.

## Criterios de Aceite

- [ ] Tela acessivel via "Regras de Alerta" em Configuracoes do Aquario
- [ ] Lista de regras com parametro, operador, threshold e severidade
- [ ] Formulario de criacao: parametro, operador (>, <, >=, <=), valor, severidade (warn/crit)
- [ ] POST /api/v1/aquariums/{id}/alert-rules para criar
- [ ] PATCH /api/v1/aquariums/{id}/alert-rules/{ruleId} para editar
- [ ] DELETE /api/v1/aquariums/{id}/alert-rules/{ruleId} para excluir
- [ ] Toggle de ativo/inativo por regra

## Notas Tecnicas

- Usar FlashList para a lista de regras
- Formulario com React Hook Form + Zod
