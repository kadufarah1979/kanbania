## Descricao

Configurar o profile de producao no EAS Build (AAB assinado para Play Store),
gerar o primeiro build de producao e criar o draft do listing na Play Store
(titulo, descricao, screenshots, icones, categoria).

## Criterios de Aceite

- [ ] `eas.json` com profile `production`: `{ "android": { "buildType": "app-bundle" } }`
- [ ] Keystore configurada via EAS Secrets (nao versionar o .jks)
- [ ] `eas build --platform android --profile production` gera AAB sem erros
- [ ] App Bundle enviado ao Internal Testing Track no Play Console
- [ ] Listing draft: titulo "AquaBook", descricao PT-BR, categoria "Ferramentas"
- [ ] Screenshots: pelo menos 2 (HomeScreen + AquariumDetail)
- [ ] Icone 512x512px e feature graphic 1024x500px

## Notas Tecnicas

- Keystore: `eas credentials` para gerar e armazenar
- versionCode deve ser incrementado a cada build
- Screenshots: usar o Z Fold 5 ou gerar com emulador de tamanho padrao (1080x1920)
