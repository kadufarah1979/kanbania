## Descricao

Tela de registro de manutencoes: troca de agua, limpeza de filtro, troca de lampada,
dosagem de produto, etc. Exibe historico com data e tipo de manutencao.

## Criterios de Aceite

- [ ] Secao "Manutencao" ou tab em AquariumDetailScreen
- [ ] Tipos: troca de agua (% volume), limpeza de filtro, troca de lampada, dosagem, outro
- [ ] Formulario: tipo, data, notas opcionais
- [ ] Historico scrollable por data
- [ ] POST /api/v1/aquariums/{id}/maintenance para registrar
- [ ] GET /api/v1/aquariums/{id}/maintenance para listar

## Notas Tecnicas

- Para troca de agua: campo adicional `volume_percent` (ex: 20%)
- Ordenar por data desc, agrupar por mes
