## Descricao

Tela de detalhe de produto do marketplace: galeria de fotos, descricao completa,
dados do seller, stock disponivel e botao "Adicionar ao Carrinho".

## Criterios de Aceite

- [ ] GET /api/v1/marketplace/listings/{id} para dados do produto
- [ ] Galeria de fotos horizontalmente scrollable com zoom
- [ ] Nome, descricao, preco, estoque disponivel
- [ ] Card do seller: avatar, nome, rating, link para perfil
- [ ] Seletor de quantidade (stepper 1-N, max = stock)
- [ ] Botao "Adicionar ao Carrinho" chama cartStore.addItem()
- [ ] Badge do carrinho atualizado no tab de navegacao

## Notas Tecnicas

- cartStore (Zustand): `{ items: CartItem[], addItem, removeItem, updateQty, clear }`
- CartItem: `{ listing_id, name, price, qty, image_url, seller_id }`
- Nao fazer POST para o backend ainda — so cartStore local (checkout faz o POST)
