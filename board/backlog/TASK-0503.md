## Descricao

Implementar o checker do PCI-DSS Req 2 no modulo `modules/req_02_checker.py`.
O Req 2 exige que sistemas nao usem configuracoes padrao de fornecedores (senhas default,
configuracoes inseguras out-of-the-box).

## Criterios de Aceite

- [ ] `modules/req_02_checker.py` implementado (nao e mais stub)
- [ ] Verifica: senhas padrao em RDS, EC2 (via SSM), security groups com portas default abertas
- [ ] Verifica: configuracoes padrao inseguras nos servicos AWS (S3 public access, etc.)
- [ ] Testes unitarios criados em `tests/test_req_02.py`
- [ ] `python main.py --stage prd --req 2` executa sem erro e retorna score
- [ ] Output inclui: status (COMPLIANT/NON_COMPLIANT/PARTIAL), findings e recomendacoes
