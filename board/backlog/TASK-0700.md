## Descricao

Tela de lista de clientes do profissional. Para cada cliente, pode ver os aquarios que o
cliente autorizou acesso (colaborador). Ver parametros, alertas e historico dos aquarios.

## Criterios de Aceite

- [ ] Tab "Clientes" no ProNavigator exibe ClientListScreen
- [ ] GET /api/v1/pro/clients para lista de clientes
- [ ] Card do cliente: nome, avatar, N aquarios vinculados, ultimo alerta
- [ ] Tap abre ClientDetailScreen com lista dos aquarios do cliente
- [ ] Ver aquario do cliente: mesma AquariumDetailScreen mas modo readonly
- [ ] Busca por nome do cliente

## Notas Tecnicas

- Permissao de colaborador: cliente convida o pro em Configuracoes do Aquario
- Pro ve os dados em tempo real via WS (mesmo hook useAquariumWS)
- Modo readonly: nao exibir controles de relay ao ver aquario de cliente
