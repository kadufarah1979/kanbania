## Descricao

Tela de central de notificacoes in-app: lista de notificacoes recebidas (alertas de
parametros, novidades, mensagens), marcacao de lida, badge de nao lidas no icone do sino.

## Criterios de Aceite

- [ ] Icone de sino no header com badge de nao lidas
- [ ] NotificationCenterScreen: lista de notificacoes com tipo, mensagem, tempo
- [ ] GET /api/v1/notifications para lista
- [ ] Tap na notificacao navega para tela relevante (AquariumDetail se alerta, etc.)
- [ ] PATCH /api/v1/notifications/{id}/read ao tocar
- [ ] Marcar todas como lidas: PATCH /api/v1/notifications/read-all
- [ ] Deletar notificacao individual com swipe
- [ ] Badge desaparece quando todas lidas

## Notas Tecnicas

- Badge count: GET /api/v1/notifications/unread-count (polling a cada 60s)
- Tipo de notificacao: `alert`, `message`, `tip`, `order_update`
