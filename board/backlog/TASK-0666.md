## Descricao

Implementar a funcionalidade de exclusao de conta conforme exigencia do Google Play Store
(Data Safety + LGPD). O usuario deve poder excluir sua conta diretamente no app.

## Criterios de Aceite

- [ ] Opcao "Excluir Conta" acessivel em ProfileScreen > Configuracoes
- [ ] Dialog de confirmacao com texto explicando o que sera deletado
- [ ] Confirmacao por texto digitado ("EXCLUIR") antes de prosseguir
- [ ] Chama `DELETE /api/v1/users/me` com sucesso
- [ ] Apos exclusao: limpar authStore + MMKV + navegar para Onboarding
- [ ] Erro de rede tratado com mensagem clara

## Notas Tecnicas

- Endpoint backend: `DELETE /api/v1/users/me` (autenticado)
- Apos exclusao, fazer `authStore.clear()` + `queryClient.clear()`
- LGPD: exibir quais dados serao removidos (perfil, aquarios, leituras, historico)
