## Descricao

Tela de gestao de consultorias: lista de consultorias (abertas, em andamento, encerradas),
criar nova consultoria para um cliente, aceitar solicitacao de cliente e encerrar com resumo.

## Criterios de Aceite

- [ ] Tab "Consultorias" no ProNavigator com lista filtrada por status
- [ ] GET /api/v1/pro/consultations para lista
- [ ] Criar consultoria: selecionar cliente, tipo (avaliacao, tratamento, manutencao), data
- [ ] Aceitar solicitacao do cliente (se cliente solicitou)
- [ ] Tela de detalhe: historico de acoes, chat, formulario de anamnese
- [ ] Encerrar consultoria: adicionar resumo + recomendacoes
- [ ] Status: pending, active, completed, cancelled

## Notas Tecnicas

- POST /api/v1/pro/consultations para criar
- PATCH /api/v1/pro/consultations/{id} para atualizar status
- Consultorias ativas aparecem no Dashboard Pro (TASK-0699)
