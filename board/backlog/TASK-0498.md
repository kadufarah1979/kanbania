## Descricao

O nginx do Frontend nao possui headers de seguranca HTTP configurados,
expondo o sistema a ataques de clickjacking, XSS e outros.
Adicionar os headers de seguranca essenciais.

## Criterios de Aceite

- [ ] `X-Frame-Options: DENY` ou `SAMEORIGIN` configurado
- [ ] `Content-Security-Policy` configurado com politica restritiva
- [ ] `Strict-Transport-Security` (HSTS) configurado (min-age 31536000)
- [ ] `X-Content-Type-Options: nosniff` configurado
- [ ] `Referrer-Policy` configurado
- [ ] Headers verificados com `curl -I https://<dominio>` ou `securityheaders.com`
- [ ] Frontend funcionando sem quebras visuais apos adicao dos headers

## Contexto Tecnico

Localizacao: `DevOps/frontend/transaction-portal/nginx/nginx.conf`
