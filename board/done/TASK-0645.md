## Descricao

Criar pacote npm `create-kanbania` para permitir instalacao via `npx create-kanbania <dir>`.
O wizard interativo pergunta owner name, timezone e agentes, gera a estrutura completa do Kanbania
(board, sprints, okrs, config.yaml, templates de agente) e opcionalmente orienta sobre o dashboard.
