## Descricao

Implementar o Feed Social (tab Comunidade no UserNavigator) com FlashList de posts,
pull-to-refresh, infinite scroll, e acoes de like e comentario.

## Criterios de Aceite

- [x] Tab "Comunidade" no UserNavigator exibe FeedScreen
- [x] FlashList com posts (avatar, nome, foto opcional, texto, parametros do aquario opcional)
- [x] Infinite scroll via cursor: GET /api/v1/social/feed?cursor=&limit=20
- [x] Pull-to-refresh
- [x] Botao de like: POST /api/v1/social/posts/{id}/like (toggle)
- [x] Contagem de likes e comentarios visivel no card
- [x] Tap no post abre PostDetailScreen com comentarios
- [x] GET /api/v1/social/posts/{id}/comments para lista de comentarios

## Implementacao

- `mobile/src/app/screens/user/FeedScreen.tsx` — FlashList + useInfiniteQuery + FAB
- `mobile/src/shared/components/PostCard.tsx` — componente reutilizavel com like optimistic update
- `mobile/src/app/screens/user/PostDetailScreen.tsx` — detalhe + lista de comentarios + input
- `mobile/src/app/navigators/UserNavigator.tsx` — ComunidadeStack com Feed/PostDetail/CreatePost
