## Descricao

Tela de gestao da fauna e flora do aquario. Lista todas as especies cadastradas com
foto, nome cientifico/popular, quantidade. Permite adicionar, editar e remover especies.

## Criterios de Aceite

- [x] FlashList de especies com foto miniatura, nome, quantidade
- [x] Adicionar especie: nome cientifico/popular, quantidade, data de adicao, observacoes
- [x] Remover com confirmacao
- [x] GET /api/v1/aquariums/{id}/livestock para lista
- [x] POST /api/v1/aquariums/{id}/livestock
- [x] DELETE /api/v1/aquariums/{id}/livestock/{speciesId}
- [x] Agrupado por categoria (peixe, coral, invertebrado, planta, outro)

## Implementacao

- `mobile/src/app/screens/user/LivestockScreen.tsx`
  - FlashList com headers de categoria + cards de habitante
  - Confirmacao de exclusao via Alert
  - FAB para AddLivestockScreen
  - Pull-to-refresh
- `mobile/src/app/screens/user/AddLivestockScreen.tsx`
  - Formulario completo (nome, cientifico, categoria picker, quantidade, data, notas)
  - expo-image-picker para foto opcional (galeria + camera)
  - multipart/form-data se tiver foto
- `mobile/src/app/navigators/UserNavigator.tsx`
  - AquariumsStack com rotas Livestock e AddLivestock
