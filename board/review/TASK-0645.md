## Descricao

Criar pacote npm `create-kanbania` para permitir instalacao via `npx create-kanbania <dir>`.
O wizard interativo pergunta owner name, timezone e agentes, gera a estrutura completa do Kanbania
(board, sprints, okrs, config.yaml, templates de agente) e opcionalmente orienta sobre o dashboard.

## Criterios de Aceite

- [ ] `npx create-kanbania meu-projeto` funciona sem instalacao previa
- [ ] Wizard interativo com prompts: owner, timezone, agentes
- [ ] Gera estrutura: `board/{backlog,todo,in-progress,review,done}/`, `sprints/`, `okrs/`, `logs/`, `projects/`
- [ ] Gera `config.yaml` preenchido com os dados do wizard
- [ ] Copia `AGENTS.md`, `CLAUDE.md` e `CODEX.md` dos templates
- [ ] Exibe instrucoes finais de uso (proximos passos)
- [ ] Funciona em Node.js 18+, sem dependencias de bash ou yq
- [ ] `package.json` com `"bin": { "create-kanbania": "./index.js" }`

## Contexto Tecnico

- Localizado em `packages/create-kanbania/` (estrutura monorepo futura)
- Dependencias: `prompts` (wizard), `fs-extra` (copy), `chalk` (output colorido)
- Convencao npm `create-*` (igual create-react-app, create-next-app)
- Nao requer publicacao no npm para testar: `node packages/create-kanbania/index.js <dir>`
