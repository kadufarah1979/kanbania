## Descrição

Implementar gráficos interativos de histórico de sensores usando Recharts.

Componentes:
- `components/dashboard/parameter-chart.tsx` — gráfico de linha para dados de sensor + pontos para manual + áreas coloridas para ranges
- `components/dashboard/time-range-picker.tsx` — seletor: 1h, 6h, 24h, 7d, 30d, custom

## Critérios de Aceite

- [ ] Gráfico de linha para leituras de sensor (contínuo)
- [ ] Pontos para leituras manuais (scatter overlay)
- [ ] Áreas coloridas horizontais para ranges (ideal verde, warn amarelo, crit vermelho)
- [ ] Tooltip com valor, timestamp e fonte
- [ ] Seletor de período (1h, 6h, 24h, 7d, 30d)
- [ ] Suporte a múltiplos parâmetros no mesmo gráfico (multi-eixo)
- [ ] Responsivo
- [ ] Integrado com API /readings (aggregation por resolução)
