## Descrição

Permitir que o usuário customize os ranges (ideal, warning, critical) dos parâmetros de cada aquário.

Endpoints:
- GET /api/v1/aquariums/{id}/parameters — listar parâmetros com ranges atuais
- PUT /api/v1/aquariums/{id}/parameters — atualizar ranges ou habilitar/desabilitar

## Critérios de Aceite

- [ ] Listar parâmetros do aquário com ranges atuais
- [ ] Atualizar ideal_min/max, warn_low/high, crit_low/high por parâmetro
- [ ] Habilitar/desabilitar parâmetros
- [ ] Validação: warn deve ser mais amplo que ideal, crit mais amplo que warn
- [ ] Apenas owner do aquário pode editar
