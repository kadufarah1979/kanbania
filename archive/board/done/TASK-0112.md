## Descricao

Atualizar a tipografia do frontend para seguir a identidade visual AquaBook:
- **Poppins** para titulos (H1, H2), logotipo, e botoes
- **Inter** para corpo de texto, dados, tabelas, menus (ja em uso)

### O que mudar

1. `frontend/src/app/layout.tsx`: importar Poppins do next/font/google
2. `frontend/tailwind.config.ts`: adicionar familia `font-display` ou `font-heading` com Poppins
3. Aplicar Poppins nos headings (h1, h2) via `@layer base` em globals.css
4. Sidebar logo text: usar Poppins Bold
5. Inter continua como font-sans (corpo)

### Nao mudar

- JetBrains Mono para codigo (manter font-mono)
- Tamanhos de fonte existentes
- Peso/estilo de elementos que nao sao headings
