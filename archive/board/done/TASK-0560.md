## Descricao

Endpoint FastAPI para buscar recomendacoes de Savings Plans por Organizational Unit (OU) via AWS Cost Explorer. Pagina React com cenarios de economia (1 ano vs 3 anos, partial vs full).

## Criterios de Aceite

- [ ] GET /api/finops/savings-plans?ou_id=... retorna recomendacoes
- [ ] Cache de 1h para evitar chamadas repetidas ao Cost Explorer
- [ ] Pagina React exibindo economia estimada por cenario
- [ ] Fallback gracioso se OU sem dados suficientes

## Contexto Tecnico

AWS Cost Explorer API: GetSavingsPlansRecommendations. Autenticacao por tenant (Bearer). Cache no backend (Redis ou in-memory). Componente React na pagina FinOps existente.
